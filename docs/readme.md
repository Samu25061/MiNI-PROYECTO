# Documentación del Proyecto

## Descripción
Este proyecto consiste en una API REST desarrollada con Node.js y Express para la gestión de items.

## Funcionalidades
- Crear items
- Obtener lista de items
- Actualizar items
- Eliminar items

## Pruebas
Se implementaron pruebas unitarias utilizando Jest para validar el correcto funcionamiento de la API.

## Integración Continua
Se configuró un pipeline CI/CD con GitHub Actions que ejecuta automáticamente:
- Instalación de dependencias
- Ejecución de pruebas

## Cobertura
Se generó un reporte de cobertura para evaluar la calidad del código.

## Estructura del Proyecto
- src/: Código fuente
- pruebas/: Pruebas unitarias
- docs/: Documentación
